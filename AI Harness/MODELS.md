# Hugging Face AI Model Task Taxonomy

## Official Hugging Face Task Categories

| Category | Purpose | Example Tasks |
|----------|---------|---------------|
| **Natural Language Processing (NLP)** | Understand and generate text | Text Generation, Text Classification, Summarization, Translation, Question Answering, Token Classification, Fill-Mask |
| **Computer Vision** | Analyze or generate images and video | Image Classification, Object Detection, Image Segmentation, Text-to-Image, Image-to-Text, Image-to-Video |
| **Audio** | Process speech and sound | Automatic Speech Recognition (ASR), Text-to-Speech (TTS), Audio Classification |
| **Multimodal** | Combine multiple modalities | Image-Text-to-Text, Visual Question Answering (VQA), Document Question Answering, Any-to-Any |
| **Tabular** | Work with structured datasets | Tabular Classification, Regression |
| **Reinforcement Learning** | Train intelligent agents | Reinforcement Learning |
